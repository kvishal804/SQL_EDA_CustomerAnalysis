Customer Analytics Project — AdventureWorks 2022
 Project Overview

This project performs end-to-end customer analytics using the AdventureWorks 2022 database.

The analysis focuses on:  

    Customer behavior  
    Retention patterns  
    Revenue contribution  
    Customer segmentation  

The project is structured into multiple phases, progressing from basic exploration to advanced segmentation and business insights.

Objectives  

      Understand customer purchasing behavior  
      Identify high-value customers  
      Analyze retention and churn patterns  
      Build business-ready datasets for Power BI  
      Deliver actionable insights through dashboards
      Project Structure
Phase 1 — Customer Overview  

    Total customers vs active customers  
    Customer acquisition trends  
    Regional distribution  
Phase 2 — Customer Behavior  

    Orders per customer  
    Average Order Value (AOV)  
    Top customers by revenue  
    Order frequency distribution  
Phase 3 — Time-Based Analysis  

    Monthly customer acquisition  
    New vs repeat customers  
    Purchase frequency (time between orders)  
Phase 4 — Cohort Analysis  

    Customer retention tracking
    Cohort-based behavior
    Retention rate calculation
Phase 5 — RFM Segmentation  

    Recency, Frequency, Monetary analysis
    Customer scoring using NTILE
    Business-driven customer segmentation
